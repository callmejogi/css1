по названию элемента(тега) : p{} - обращение ко всем элементам с тегом p


по id: #face{} - обращение ко всем элементам с id="face"


по классу: .vibe{} - обращение ко всем элементам с class="vibe"


мультиклассы: .vibe.level1{} - обращение ко всем элементам с class="vibe class1"


по вложенности: div p {} - в этом случае стиль будет применяться к элементу

, когда соблюдается следующий код


дочерний селектор: li > a{} - стиль применяется к элементу a, только когда он является дочерним для элемента li(то есть находится внутри li).


родственные селекторы: h1~p{} - стиль применится ко всем элементам с тегом

,лежащим после h1 при условии ,что оба тега находятся в одном контейнере(общий родитель)


соседние селекторы: b+i{} - Стиль при такой записи применяется к элементу i, но только в том случае, если он является соседним для элемента b и следует сразу после него.


по атрибуту: [title] - Стиль применяется к тем элементам, внутри которых добавлен указанный атрибут


[атрибут="<значение>"] ; E[атрибут="<значение>"] - В первом случае стиль применяется ко всем тегам, которые содержат указанное значение атрибута. А во втором — только к определённым селекторам.


универсальный селектор: * - используется ,когда требуется установить одновременно один стиль для всех элементов веб-страницы
